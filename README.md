# colintinney.com
> Personal website.

Built with love, but mostly Jekyll, HTML, and SCSS.

## Developing

Requirements:

* [Jekyll](https://jekyllrb.com/docs/installation/)

Running (watch mode):

```
jekyll serve
```

Building:

```
jekyll build
```
