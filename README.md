## Getting Started

### Requirements

* Node.js - Latest LTS release

### Installing

To install dependencies:

```
$ npm install
```

### Running

To run a development server with live-reload:

```
$ npm run develop
```

### Building

To build the website for production:

```
$ npm run build
```

Build output is located within the `public` folder.
